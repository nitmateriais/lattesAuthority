# lattesAuthority
Serviço REST para alimentar uma autoridade de nomes baseada no Currículo Lattes. 


## Instalando

```
sudo apt-get install python-tornado python-lxml python-suds python-repoze.lru
sudo useradd -r restpy
sudo make install
```
